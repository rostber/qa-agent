# QA Agent CLI

Интерактивная CLI-утилита для общения с AI-агентом с поддержкой MCP сервера Playwright.

## Быстрый старт

```bash
# Установка
npm install
npx playwright install

# Настройка (создайте .env файл)
cp .env.example .env

# Запуск
npm start
```

## Конфигурация

Создайте файл `.env` с параметрами:

```env
OPENAI_API_TOKEN=your_token
OPENAI_BASE_URL=http://localhost:11434/v1
OPENAI_MODEL=qwen
```

## Функционал

### Управление браузером

| Команда | Описание |
|---------|----------|
| `/open <url>` | Открыть браузер по адресу для ручной авторизации |
| `/prompt <текст>` | Изменить системный промт агента |
| `/exit` или `/quit` | Выйти из программы |

### Ввод сообщений

- **Однострочный ввод:** введите текст и нажмите Enter
- **Многострочный ввод:** нажмите Enter после каждой строки, пустая строка отправляет сообщение

### Работа агента

1. Откройте браузер командой `/open <url>`
2. Выполните авторизацию вручную в открывшемся окне
3. Давайте агенту задачи на естественном языке

## Доступные инструменты

Агент может использовать следующие инструменты Playwright MCP:

| Инструмент | Описание |
|------------|----------|
| `browser_navigate` | Перейти по URL |
| `browser_click` | Кликнуть по элементу (по тексту или селектору) |
| `browser_type` | Ввести текст в поле |
| `browser_fill_form` | Заполнить форму несколькими полями |
| `browser_snapshot` | Получить доступность страницы |
| `browser_take_screenshot` | Сделать скриншот |
| `browser_wait_for` | Ожидать появления текста |
| `browser_press_key` | Нажать клавишу |
| `browser_hover` | Навести курсор |
| `browser_select_option` | Выбрать опцию в выпадающем списке |
| `browser_drag` | Перетащить элемент |

## Поведение агента

- **Последовательное выполнение:** агент выполняет шаги один за другим
- **Проверка ошибок:** если шаг завершается ошибкой, выполнение прекращается
- **Отчёт о результате:** после выполнения всех шагов агент выводит итоговый отчёт
- **Глобальная сессия:** браузер остаётся открытым между запросами
